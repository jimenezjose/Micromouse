# Micromouse Simulator

## [Wiki](https://github.com/jimenezjose/Micromouse_Simulator/wiki)

## How to Run Micromouse Simulator:
```
$ git clone https://github.com/jimenezjose/Micromouse_Simulator.git
$ cd Micromouse_Simulator
$ ./runSimulator.sh
```
